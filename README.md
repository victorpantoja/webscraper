webscraper
==========

Just an easy way to get Facebook and Twitter Profiles

DEPENDENCIES
------------

After instaling, please make sure you have the following dependencies:
- Python 2.7+
- beanstalkd: http://kr.github.io/beanstalkd/download.html


INSTALING
---------
$ mkvirtualenv webscraper
$ make setup


RUNNING
------------
$ make start


TESTING
----------
$ make test
